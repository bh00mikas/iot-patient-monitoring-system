# IoT Patient Monitoring System

## Overview
This project simulates a patient health monitoring system using Arduino and IoT sensors. 
The system collects physiological parameters such as temperature and heart rate and 
displays them in real time.

## Features
- Real-time sensor monitoring
- Arduino-based embedded system
- Serial output for patient data
- Tinkercad circuit simulation

## Components Used
- Arduino Uno
- Temperature Sensor
- Pulse Sensor
- LCD Display
- Breadboard
- Resistors

## How it Works
Sensors collect patient data and send signals to the Arduino. 
The microcontroller processes these signals and outputs the results 
to the serial monitor or display module.
